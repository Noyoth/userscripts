# Userscripts

## Description

Personal collection of original userscripts designed to enhance web experience. 

## Script List

| Script Name | Functional Description | Installation |
| :--- | :--- | :--- |
| Bilibili in One Tab | Forces all Bilibili links and search results to open in the current tab. | [Install](https://greasyfork.org/en/scripts/565407-bilibili-in-one-tab) |
| Open in IINA | Adds a keyboard shortcut (Option + I) to open the current page URL in IINA. | [Install](https://greasyfork.org/en/scripts/565408-open-in-iina) |
